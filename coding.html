Block Coding 系列课程

知行教育Block Coding系列课程是为2030年的世界准备的教育，让学生通过使用Blocks快乐地学习编程，在学习的过程中提高自己的逻辑思维能力，独立思考的好习惯和丰富的创造力。课程主要由3个模块组成，分别为Block CodeMao，Block Python和Block Sandbox。我们通过创造性地教学，让学生通过编程独立做出有趣实用的项目，并可以和全世界的用户一起分享自己的作品。所有课堂上的课件、项目和教材均由我们自主研发，确保每一节课都是高质量的精品课程。

优势：打破传统枯燥的编程教学，用最简单的方法去解决最复杂的问题。培养编程的兴趣，让学生感受编程的魅力。

亮点：最适合初学者入门的编程课程，无需任何编程基础，适用于任何年龄段的学生，轻松做出电脑软件和手机小程序。

Block CodeMao

课程内容: 学习Block Coding, 自己独立作出有趣的游戏和软件，并可以将制作的成果做成二维码链接在微信和朋友圈分享，让世界各地的亲朋好友都能体验一下自己的作品。

课程目标: 激发学员对编程的兴趣，让0基础的学员找到正确有趣的编程学习路径，做出有趣的作品。

Block Python

课程内容：通过使用Block快速上手当今最流行的编程语言Python，以解决数学几何问题的方式去学习Python，让学员学习编程的同时又可以学习数学，不仅可以提高思维逻辑而且可以提高数学几何能力。

课程目标: 用编程猫学习当今最火的编程语言Python，拥抱人工智能，和AI一起改变世界。

Block Sandbox

课程内容: 在原有编程的基础上进行沙盒创作，使用Block创造出整个沙盒世界。

课程目标: 通过3D建模，无限激发学生的创造力，做出独一无二优秀的作品。

成人Python：

当今社会白领，前台人员，HR，管理人员等的工作量特别多，有时候处理一个表格，写一段文字，做一个ppt都需要花费一整天的时间。我们根据当今社会的需求，研发了一套专门针对成人的Python编程课，不需要任何的基础，让0基础的小白快速上手，并且将所学的知识用于实践比如用Python一键处理excel表格，多篇幅处理word文档，快速整合ppt文件等。我们还会介绍更有趣的知识比如如何用Python去p图，分析微信好友的数据等等。

课程亮点：有趣 实用 高效

适合对象：在职场中想要高效办公的白领，对自动化办公感兴趣的0基础小白，对Python感兴趣的成人。

CodeMao Level 1 - Age: G2 ~ G6

(* No programming experience required)
You will learn basic block coding skills like keyboard events, loop, condition, clones and variables, etc. You will have the ability to build your small games.

CodeMao Level 2 - Age: G2 ~ G6

(* Basic Scratch or Codemao skill required)
You will use professional api like physical engine, AI engine and VR engine to build your own games. You will be able to develop your own creative games.

CodeMao Level 3 - Age: G2 ~ G6

(* Basic Scratch or Codemao skill required)
You will learn how to collect sprites online to make your own professional game. You are able to design, code and test the project by yourself.

Python Level 1 - Age: G7 ~ G10

(* No programming experience required)
You will learn basic coding skill in Python through turtle graphic. You will understand data type, variable, list, dictionary, for loop, while loop, function and so much more.

Python Level 2 - Age: G7 ~ G12

(* Basic programming knowledge required)
You will learn advanced programming topcis like recursion, data structure, algorithm and so on. You will be able to use Python to solve real math question. In addition, this section will be covered some programming competition questions.

Python Level 3 - Age: G10 ~ Undergraduate

(* Basic programming knowledge required)
You will learn professional Python skills like data mining, image processing, AI, etc. You will be able to use Python to analyse WeChat data, manage office documents(word, excel, ppt, pdf), analyse stock data, modify image and so much more.

Introduction to Web Development - Age: G7 ~ G12

(* No programming experience required)
You will discover the Internet and understand how the Internet work. You will learn HTML, CSS, JavaScript and jQuery and be able to build your own website and some interesting web applications.

Advanced Web Development - Age: G7 ~ G12

(* HTML, CSS & JavaScript experience required)
You will go deeper in CSS and JavaScript to build many interesting web projects to improve your web programming skills. In addition, you will learn some professional skills like api, more js libraries and ES6 to perform like a real web developer.

Professional Web Development - Age: G10 ~ Undergraduate

(* Advanced web programming skill required)
You are going to be a professional developer. You will learn Bootstrap, Sass and JavaScript latest library like ReactJS or Vue.js. You will be able to build commercial websites and professional web applications.

Programming Competition - Age: G7 ~ G12

(* Basic programming skill required)
You will learn algorithm and data sctructre in order to prepare for programming competition. You will learn some university level programming and math knowledge.
